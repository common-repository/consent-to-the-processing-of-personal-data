To agree to the processing of personal data in the form of commenting

== Description ==

The plugin adds a tick "Consent to the processing of personal data" in the standard comment form wordpress.

In connection with the change in the legislation of the Russian Federation (https://www.consultant.ru/document/cons_doc_LAW_61801/ - up-to-date version
Law on Consultant +) - obtaining consent to the processing of personal data from the user is
mandatory for each operator of personal data. For more information, please click the link -
https://rkn.gov.ru/treatments/p459/p468/.


== Installation ==

1. Upload the zip file and unzip it in the /wp-content/plugins/ directory
2. Activate the plug-in on the "Plugins" page in the WordPress control panel.
